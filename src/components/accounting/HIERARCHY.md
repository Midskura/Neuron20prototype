## Quick Reference: Common Patterns

### Page Header Pattern
```tsx
<div className="mb-6">
  <h1 className="text-[#0A1D4D] mb-2">Page Title</h1>
  <p className="text-[14px] text-[#6B7280] leading-[20px]">
    Page description text
  </p>
</div>
```

### Table Container Pattern
```tsx
<div 
  className="border border-[#E5E7EB] overflow-hidden" 
  style={{ borderRadius: 'var(--radius-sm)' }}
>
  <Table>
    {/* Table content */}
  </Table>
</div>
```

### Two-Column Form Pattern
```tsx
<div className="grid grid-cols-2 gap-6">
  <div className="space-y-4">
    {/* Left column fields */}
  </div>
  <div className="space-y-4">
    {/* Right column fields */}
  </div>
</div>
```

---

This hierarchy ensures a **consistent, scalable, and maintainable** structure for the entire Accounting module.
