## Key Features:

### 1. **Data Source Cards** (Top)
- Visual card selector for: Quotations, Customers, Contacts, Activities, Budget Requests
- Selected card is highlighted with teal accent
- Icons matching each data type

### 2. **Dynamic Filter Bar** (Expandable/Collapsible)
- **"+ Add Filter"** dropdown with ALL available fields:
  - Text fields: Customer Name, Contact Name, Service Type, Origin, Destination
  - Date fields: Created Date, Updated Date, Valid Until
  - Number fields: Amount, Quantity
  - Status fields: Status, Priority, Stage
  - User fields: Assigned To, Created By
  
- **Each filter row** has:
  - Field selector dropdown (left)
  - Operator dropdown (is, is not, contains, between, >=, <=, etc.)
  - Value input (text/dropdown/date picker based on field type)
  - Remove [Ã—] button

- **"Hide Filters"** link to collapse entire section

### 3. **Column Selector**
- Checkbox list to show/hide columns
- Drag to reorder columns

### 4. **Live Results Table**
- Updates immediately as filters change
- Shows row count
- Sortable columns
- Matches Inquiries table design

### 5. **Export & Save**
- Export buttons (CSV, Excel, PDF) in header
- "Save Report Configuration" button at bottom to save filter setup
